# InfiniteOS

InfiniteOS 是面向云、边、端三侧的统一操作系统，旨在用开源 OS 连接云、边、端算力，让 AI 运行无处不在。一套 InfiniteOS 适用于云端、边缘与端侧场景，通过统一架构实现三侧算力的无缝连接与协同。

## 项目概述

InfiniteOS 采用统一架构设计，在云、边、端任一场景下提供一致的操作系统能力。无论是云端大规模计算、边缘实时处理还是端侧轻量推理，均可使用同一套 InfiniteOS，降低运维与迁移成本。

## 镜像与硬件适配

InfiniteOS 提供**按硬件架构区分的镜像版本**，便于在不同设备上部署：

- **x86 / x86_64**：适用于主流服务器、PC 及部分边缘设备
- **ARM / aarch64**：适用于 ARM 服务器、开发板、嵌入式及端侧设备
- 其他架构的镜像将随社区需求持续补充

请根据您的目标硬件在 [GitHub Releases](https://github.com/ainet-os/InfiniteOS/releases) 中选择对应架构的镜像进行安装。

## 技术特点

### 统一架构

- **一套系统，多侧部署**：同一套 InfiniteOS 可部署在云、边、端
- **API 一致**：各侧提供一致的 API 与使用体验
- **应用可移植**：应用可在云、边、端之间迁移与复用

### 算力协同

- **动态调度**：根据任务需求分配算力
- **负载均衡**：优化资源利用与响应延迟
- **故障转移**：自动检测与恢复，提升可用性

### 开源开放

InfiniteOS 完全开源，采用 Apache 2.0 许可证，代码在 GitHub 公开，欢迎社区参与贡献。

## 快速开始

### 下载与安装

根据您的硬件架构选择对应镜像（如 x86、ARM）：

```bash
# 示例：x86_64 镜像
wget https://github.com/ainet-os/InfiniteOS/releases/latest/download/infiniteos-x86_64.iso

# 或 ARM64 镜像
wget https://github.com/ainet-os/InfiniteOS/releases/latest/download/infiniteos-aarch64.iso
```

使用工具（如 `dd` 或 Rufus）将 ISO 写入安装介质，按安装向导完成安装。具体镜像文件名以 [Releases](https://github.com/ainet-os/InfiniteOS/releases) 页面为准。
