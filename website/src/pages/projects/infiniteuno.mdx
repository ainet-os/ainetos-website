# InfiniteUno

InfiniteUno 是云边端算力协同平台，提供统一的算力管理、调度与协同能力，让 AI 应用在云、边、端之间自由流动。InfiniteUno 提供三个版本，满足不同使用场景与规模需求。

## 版本说明

### InfiniteUno CE（社区版）

- **开源免费**：代码完全开放，采用 Apache 2.0 许可证
- **自建部署**：可在自有环境中部署与二次开发
- **适合**：个人开发者、小团队、技术尝鲜与社区贡献

### InfiniteUno SaaS（SaaS 版）

- **开箱即用**：无需自建基础设施，通过云端服务使用
- **按需使用**：按使用量或订阅方式使用算力协同能力
- **适合**：希望快速上手的团队与中小规模业务

### InfiniteUno EE（企业版）

- **企业级能力**：高可用、安全合规、专属支持与定制化能力
- **商业授权**：面向企业客户的商业版本
- **适合**：对稳定性、安全与支持有更高要求的企业场景

## 项目概述（以 CE 为例）

InfiniteUno CE 是社区版，旨在解决云、边、端三侧算力资源的管理与协同问题。通过 InfiniteUno，您可以：

- 统一管理云、边、端三侧的算力资源
- 智能调度任务到最合适的计算节点
- 实现 AI 应用的动态部署和迁移
- 监控和管理整个算力集群

## 核心功能

### 统一算力管理

- **资源发现**：自动发现和注册云、边、端节点
- **资源监控**：实时监控各节点资源使用情况
- **资源分配**：灵活配置和分配算力资源
- **资源池化**：将分散算力整合为统一资源池

### 智能调度

- **负载均衡**：根据节点负载智能分配任务
- **延迟优化**：优先调度到延迟更低的节点
- **成本优化**：在满足性能前提下优化成本
- **故障转移**：自动检测故障并迁移任务

### AI 应用支持

- **模型部署**：支持各类 AI 模型的快速部署
- **动态迁移**：根据负载动态迁移推理任务
- **边缘推理**：支持将推理下沉到边缘节点
- **云端训练**：支持云端大规模模型训练

### 监控和运维

- **实时监控**：系统状态与性能指标
- **日志管理**：集中管理与查询日志
- **告警通知**：多种告警与通知渠道
- **性能分析**：辅助优化系统性能

## 架构设计（CE）

InfiniteUno CE 采用微服务架构，主要组件包括：

- **控制平面**：资源管理与任务调度
- **数据平面**：计算任务执行
- **监控组件**：系统监控与指标收集
- **API 网关**：统一 API 接口

## 快速开始（CE）

### 安装 InfiniteUno CE

```bash
git clone https://github.com/ainet-os/InfiniteUno-CE.git
cd infinite-uno-ce
docker-compose up -d
```

### 配置节点

```bash
# 注册云端节点
infinite-uno register --type cloud --endpoint https://cloud.example.com

# 注册边缘节点
infinite-uno register --type edge --endpoint https://edge.example.com

# 注册端侧节点
infinite-uno register --type end --endpoint https://end.example.com
```

### 部署应用

```bash
infinite-uno deploy --name ai-inference \
  --image ai-model:latest \
  --replicas 3 \
  --strategy edge-first
```

## 与 InfiniteOS 集成

InfiniteUno 与 InfiniteOS 深度集成：在 InfiniteOS（云、边、端任一架构）上运行 InfiniteUno 可获得一致体验，并通过 InfiniteUno 统一管理所有 InfiniteOS 节点，实现云边端无缝协同。
